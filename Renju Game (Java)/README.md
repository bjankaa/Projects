# Renju

This project is a Java implementation of the Renju board game, a strategic two-player game played on a 15×15 board. The application provides a graphical interface where two players can play the game.

The goal of the project was to design and implement a complete board game application using object-oriented programming in Java, including the game logic, rule handling, and graphical user interface.

## Features

- Graphical user interface for gameplay
- Turn-based two-player gameplay
- Stone placement on a 15×15 board
- Game rule explanation available from the menu
- Scoreboard displaying results of previous games
- Pause menu with navigation options
- Persistent storage of game results in a text file

## Technologies Used

- **Java**
- **Object-Oriented Programming (OOP)**
- **Graphical User Interface (GUI)**
