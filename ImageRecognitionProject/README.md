# Training an Image Recognition Model on a Generated Dataset

This project extends a previous implementation of the AlexNet convolutional neural network by exploring how generated image data can be used for training image recognition models.

The goal of the project was to investigate whether a dataset generated through image generation techniques can be effectively used to train a convolutional neural network and how the results compare to models trained on real datasets.

To evaluate the effectiveness of synthetic training data, the model's performance was tested on both, the generated image dataset and the CIFAR-10 dataset.

The performance of the model trained on the generated dataset was compared with the results of the previously trained model that used the CIFAR-10 dataset.

## Features

- Generation of a synthetic image dataset
- Training a CNN model using the generated dataset
- Testing the trained model on both generated and CIFAR-10 images
- Comparison with the previous AlexNet model trained on CIFAR-10
- Analysis of the effectiveness of synthetic training data comp
