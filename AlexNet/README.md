# AlexNet Image Recognition Implementation

This project implements the AlexNet CNN architecture for image classification using the CIFAR-10 dataset. The goal of the project was to study and demonstrate how a well-known deep learning architecture can be applied to a real-world image recognition task.

## Features

- Implementation of the AlexNet CNN architecture
- Training the model on the CIFAR-10 dataset
- Evaluation of the model's performance on image classification tasks
- Detailed explanations of the implemented code and model behavior

## Technologies Used

- **Python**
- **Deep Learning / Convolutional Neural Networks**
- **CIFAR-10 Dataset**
